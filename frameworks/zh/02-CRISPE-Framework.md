# CRISPE 框架 (CRISPE Framework)

> 来源：[Juuzt AI - CRISPE Framework](https://juuzt.ai/knowledge-base/prompt-frameworks/the-crispe-framework/)

## 框架概述

CRISPE 框架（Clarity, Relevance, Iteration, Specificity, Parameters, Examples）是一种突破性的 AI 提示设计方法论，旨在实现最大精确度和相关性。通过将清晰度与迭代优化相结合，CRISPE 确保与大语言模型（LLM）的每次交互都能产出有影响力和可执行的输出。

---

## 六大核心组件

| 组件 | 英文 | 说明 |
|------|------|------|
| **清晰度** | Clarity | 清楚定义任务或目标，最小化歧义 |
| **相关性** | Relevance | 使提示与上下文或受众对齐以获得准确结果 |
| **迭代** | Iteration | 基于反馈进行迭代改进，优化响应 |
| **具体性** | Specificity | 添加精确细节或约束来缩小 AI 的焦点 |
| **实用性** | Practicality | 确保输出可操作且可实施 |
| **示例** | Examples | 提供样本输入或输出来引导 AI 达到期望结果 |

---

## CRISPE vs RACEF 对比

| 方面 | CRISPE | RACEF |
|------|--------|-------|
| **聚焦点** | 清晰度和结构化精确 | 灵活性和迭代改进 |
| **最适合** | 技术性和可重复的任务 | 探索性或不断演变的挑战 |
| **灵活性** | 适合结构化输出 | 更适合动态、上下文驱动的任务 |

**结论**：选择 CRISPE 用于可预测、目标导向的结果；选择 RACEF 用于更具适应性的任务。

## 提示示例

### 示例1：电商平台营销活动

**场景**：为电商平台创建营销活动

```
Clarity（清晰度）: 开发一个促进季节性促销销售的活动。
Relevance（相关性）: 聚焦目标25-35岁的年轻专业人士。
Iteration（迭代）: 在社交媒体平台上测试不同的广告格式。
Specificity（具体性）: 包含有效期两周的20%折扣码。
Examples（示例）: 参考服装行业类似的成功活动案例。
```

### 示例2：员工培训项目

**场景**：为客服人员设计培训项目

```
Clarity（清晰度）: 创建一个提升客户互动技能的培训项目。
Relevance（相关性）: 聚焦处理高压情况和投诉。
Iteration（迭代）: 包括角色扮演场景来测试不同方法。
Specificity（具体性）: 为电子行业的在线零售商量身定制内容。
Examples（示例）: 参考领先电商公司使用的成熟方法。
```

### 示例3：产品功能优先级

**场景**：为新软件产品确定功能优先级

```
Clarity（清晰度）: 识别与产品核心价值主张一致的功能。
Relevance（相关性）: 聚焦能提高用户参与度和留存率的功能。
Iteration（迭代）: 通过多次Beta测试周期评估用户反馈。
Specificity（具体性）: 突出针对中小企业的功能。
Examples（示例）: 参考竞争对手产品以识别差距。
```

### 示例4：内容创作策略

**场景**：为旅游博客创建内容策略

```
Clarity（清晰度）: 制定增加博客读者量的策略。
Relevance（相关性）: 聚焦生态旅游和可持续旅行主题。
Iteration（迭代）: 尝试不同内容格式，如视频、信息图和长文。
Specificity（具体性）: 目标为18-30岁重视可持续旅行的读者。
Examples（示例）: 参考具有类似主题的成功旅游博客。
```

### 示例5：企业可持续发展计划

**场景**：为制造公司创建可持续发展计划

```
Clarity（清晰度）: 设计一个减少公司碳足迹的可持续发展计划。
Relevance（相关性）: 聚焦可再生能源和废物减少举措。
Iteration（迭代）: 在大规模实施前测试试点项目。
Specificity（具体性）: 为中型制造公司量身定制计划。
Examples（示例）: 参考行业内成功的可持续发展策略。
```

### 示例6：客户保留策略

**场景**：为 SaaS 平台制定客户保留策略

```
Clarity（清晰度）: 制定减少现有订阅用户流失的策略。
Relevance（相关性）: 聚焦提供个性化用户体验。
Iteration（迭代）: 测试多种互动方法，如邮件活动和应用内通知。
Specificity（具体性）: 包含忠诚度奖励计划以激励长期订阅。
Examples（示例）: 参考顶级 SaaS 公司使用的保留策略。
```

## 最佳配合的生成式 AI 模型

| 模型 | 优势 |
|------|------|
| **OpenAI GPT-4** | 非常适合需要高清晰度和细节的任务，如技术文档或分步指南 |
| **Google Bard/Gemini** | 适合创建特定受众内容或优化创意想法 |
| **Anthropic Claude** | 适合迭代过程和以相关性为焦点的总结任务 |
| **Meta Llama 2** | 适合需要结构化精确的特定领域、技术或行业聚焦项目 |

---

## 框架优缺点

### ✅ 优势

- **全面性**：涵盖有效提示设计的所有方面，从清晰度到迭代
- **灵活性**：适应创意、分析和技术任务
- **精确性**：通过详细参数确保目标化、可执行的输出

### ⚠️ 劣势

- **耗时**：为简单任务制作包含所有 CRISPE 元素的提示可能需要更长时间
- **过度具体化**：过于狭窄的参数可能限制 AI 的创造潜力

## 适用场景

CRISPE 框架非常适合需要精确性、相关性和适应性的任务：

- 📊 战略规划和执行
- 📚 教育内容开发
- 📈 数据分析和报告
- 🎨 创意头脑风暴
- ⚙️ 企业环境中的流程优化

---

## 总结

CRISPE 框架是掌握 AI 提示工程的强大工具。它对清晰度、具体性和迭代优化的强调确保用户在保持创意探索灵活性的同时获得精确和相关的输出。无论你是在制定营销策略、撰写研究论文还是进行分析，CRISPE 都能提供在 AI 驱动任务中取得成功所需的结构和深度。
