# CRISPE Framework

> Source: [Juuzt AI - CRISPE Framework](https://juuzt.ai/knowledge-base/prompt-frameworks/the-crispe-framework/)

## Framework Overview

The CRISPE Framework (Clarity, Relevance, Iteration, Specificity, Parameters, Examples) is a groundbreaking methodology for designing AI prompts that achieve maximum precision and relevance. By blending clarity with iterative refinement, CRISPE ensures that every interaction with large language models delivers impactful and actionable outputs.

---

## Six Core Components

| Component | Description |
|-----------|-------------|
| **Clarity** | Clearly define the task or goal to minimize ambiguity |
| **Relevance** | Focus on aligning the prompt with the context or audience for accurate results |
| **Iteration** | Incorporate iterative improvements by refining responses based on feedback |
| **Specificity** | Add precise details or constraints to narrow the AI's focus |
| **Practicality** | Ensure that the output is actionable and feasible for implementation |
| **Examples** | Provide sample inputs or outputs to guide the AI towards the desired outcome |

---

## CRISPE vs RACEF Comparison

| Aspect | CRISPE | RACEF |
|--------|--------|-------|
| **Focus** | Clarity and structured precision | Flexibility and iterative improvement |
| **Best for** | Technical and repeatable tasks | Exploratory or evolving challenges |
| **Flexibility** | Great for structured outputs | Better for dynamic, context-driven tasks |

## Prompt Examples

### Example 1: E-commerce Marketing Campaign

**Scenario**: Creating a marketing campaign for an e-commerce platform

```
Clarity: Develop a campaign to boost sales for a seasonal promotion.
Relevance: Focus on targeting young professionals aged 25-35.
Iteration: Test different ad formats across social media platforms.
Specificity: Include a 20% discount code valid for two weeks.
Examples: Reference similar successful campaigns in the apparel industry.
```

### Example 2: Employee Training Program

**Scenario**: Designing a training program for customer support staff

```
Clarity: Create a training program to enhance customer interaction skills.
Relevance: Focus on handling high-stress situations and complaints.
Iteration: Include role-playing scenarios to test different approaches.
Specificity: Tailor the content for online retailers in the electronics industry.
Examples: Reference proven methods used by leading e-commerce companies.
```

### Example 3: Content Creation Strategy

**Scenario**: Creating a content strategy for a travel blog

```
Clarity: Develop a strategy to increase blog readership.
Relevance: Focus on eco-tourism and sustainable travel topics.
Iteration: Experiment with different content formats such as videos, infographics, and long-form articles.
Specificity: Target readers aged 18-30 who value sustainable travel.
Examples: Reference successful travel blogs with similar themes.
```

### Example 4: Corporate Sustainability Plan

**Scenario**: Creating a sustainability plan for a manufacturing company

```
Clarity: Design a sustainability plan to reduce the company's carbon footprint.
Relevance: Focus on renewable energy and waste reduction initiatives.
Iteration: Test pilot programs before implementing at scale.
Specificity: Tailor the plan for mid-sized manufacturing companies.
Examples: Reference successful sustainability strategies from the industry.
```
