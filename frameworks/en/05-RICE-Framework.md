# RICE Framework

> Source: [Juuzt AI - RICE Framework](https://juuzt.ai/knowledge-base/prompt-frameworks/the-rice-framework/)

## Framework Overview

The RICE Framework is a powerful prioritization tool that brings clarity and precision to decision-making in AI prompt engineering. By evaluating options through four core components‚ÄîReach, Impact, Confidence, and Effort‚Äîthis framework enables users to identify high-value opportunities with minimal resources.

---

## Four Core Components

| Component | Description |
|-----------|-------------|
| **Reach** | Measure how many people or systems will benefit from the change |
| **Impact** | Assess the potential value or improvement the change will bring |
| **Confidence** | Evaluate how certain you are that the idea will succeed |
| **Effort** | Estimate the resources (time, money, manpower) required |

### RICE Score Formula

```
RICE Score = (Reach √ó Impact √ó Confidence) / Effort
```

## Prompt Examples

### Example 1: SaaS Feature Prioritization

**Scenario**: Deciding which features to prioritize for a SaaS product

```
Role: As a product manager, propose a feature prioritization strategy for a SaaS product.
Steps: Score each feature based on Reach, Impact, Confidence, and Effort, then calculate the RICE score.
Outcome: Present a prioritized feature list based on RICE scores with justifications.
```

### Example 2: Marketing Campaign Planning

**Scenario**: Creating a marketing campaign for a new product

```
Role: As a marketing strategist, design a campaign for a new eco-friendly product.
Steps: Evaluate campaign ideas for Reach, Impact, Confidence, and Effort to calculate their RICE scores.
Outcome: Recommend the top campaign with a plan for execution.
```
