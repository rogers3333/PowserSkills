# RELIC Framework

> Source: [Juuzt AI - RELIC Framework](https://juuzt.ai/knowledge-base/prompt-frameworks/the-relic-framework/)

## Framework Overview

The RELIC Framework is a structured prompt engineering tool designed to optimize interactions with ChatGPT. By focusing on five essential components‚ÄîRole, Emphasis, Limitation, Information, and Challenge‚ÄîRELIC helps users craft clear, focused, and goal-oriented prompts.

---

## Five Core Components

| Component | Description |
|-----------|-------------|
| **Role** | Define ChatGPT's role in the interaction (advisor, strategist, creator, etc.) |
| **Emphasis** | Highlight the most important aspects of your request |
| **Limitation** | Set boundaries to keep the response concise and relevant |
| **Information** | Provide the necessary background or context |
| **Challenge** | Define the problem or task with clear objectives |

## Prompt Examples

### Example 1: Blog Post Outline

**Scenario**: Outlining a blog post about eco-friendly travel tips

```
Role: As a travel blogger, focus on practical tips for sustainable travel.
Emphasis: Beginner-friendly advice.
Limitation: Limit content to beginner-friendly advice.
Information: Include examples of eco-friendly destinations and travel methods.
Challenge: Create a detailed blog outline with actionable advice and captivating subheadings.
```

### Example 2: SEO Optimized Blog Post

**Scenario**: Creating an SEO-optimized blog post for a travel website

```
Role: As an SEO specialist, emphasize trending keywords and engaging content.
Limitation: Ensure the post is no longer than 1200 words.
Information: The topic is 'Top 10 Hidden Gems in Europe.'
Challenge: Create a draft that ranks on the first page of search results.
```

### Example 3: Sales Email Campaign

**Scenario**: Designing a sales email campaign for a new product

```
Role: As a marketing strategist, focus on crafting personalized and engaging email content.
Limitation: Ensure each email is under 150 words.
Information: The product is a subscription-based meal planning service.
Challenge: Increase click-through rates by 20%.
```

### Example 4: Online Course Development

**Scenario**: Designing an online course for professionals

```
Role: As an instructional designer, prioritize clarity and engagement.
Limitation: Ensure each module is under 20 minutes.
Information: The course is on 'Mastering Project Management Tools.'
Challenge: Create a course outline that maximizes learner retention.
```

### Example 5: Investor Pitch

**Scenario**: Creating a pitch deck for potential investors

```
Role: As a startup founder, highlight key metrics and growth potential.
Limitation: Keep the deck to 10 slides.
Information: The startup focuses on AI-driven healthcare solutions.
Challenge: Attract $1M in seed funding.
```
