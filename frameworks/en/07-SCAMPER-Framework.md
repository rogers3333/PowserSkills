# SCAMPER Framework

> Source: [Juuzt AI - SCAMPER Framework](https://juuzt.ai/knowledge-base/prompt-frameworks/the-scamper-framework/)

## Framework Overview

The SCAMPER Framework is a powerful tool for brainstorming and problem-solving that leverages seven creative strategies‚ÄîSubstitute, Combine, Adapt, Modify, Put to another use, Eliminate, and Reverse. By breaking down challenges and generating fresh ideas, SCAMPER empowers users to approach innovation with confidence.

---

## Seven Core Strategies

| Strategy | Description |
|----------|-------------|
| **Substitute** | Replace one element of an idea or process with another |
| **Combine** | Merge multiple concepts to create something new |
| **Adapt** | Modify an idea to fit different needs or environments |
| **Modify** | Alter or magnify elements to enhance value |
| **Put to Another Use** | Repurpose an idea or tool for a new application |
| **Eliminate** | Remove unnecessary parts to streamline |
| **Reverse** | Reimagine an idea from an opposite perspective |

## Prompt Examples

### Example 1: Product Development

**Scenario**: Improving a sustainable water bottle

```
Substitute: Replace plastic with bamboo or glass.
Combine: Add a built-in filter system.
Adapt: Make the design suitable for outdoor enthusiasts.
Modify: Add a liquid level indicator.
Put to Another Use: Allow it to double as a snack container.
Eliminate: Remove unnecessary features to reduce weight.
Reverse: Create a collapsible version for easy transport.
```
